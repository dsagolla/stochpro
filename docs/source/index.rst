Install
-------

Install using either of the following two methods.

1. Install from PyPi
~~~~~~~~~~~~~~~~~~~~

.. |pypi| |pyversions| |format| 

The ``stochpro`` package is available on
`PyPi <https://pypi.org/project/stochpro/>`_ and can be installed using ``pip``

.. code-block:: bash

    pip install stochpro


2. Install from Source
~~~~~~~~~~~~~~~~~~~~~~

.. |release| |licence|

Install directly from the source code by

.. code-block:: bash

	git clone https://github.com/dsagolla/stochpro.git
	cd stochpro
	pip install .
